フロントエンドエンジニアとして6年、React / Next.js / TypeScript を中心に大規模Webアプリケーション開発に従事してきました。実装だけでなく、要件定義・仕様策定といった上流工程から参画し、PM・バックエンド・デザイナーと連携しながらプロジェクトを推進してきた経験があります。

近年は Datadog を活用した障害調査・品質改善、GitHub Actions による CI/CD 整備、レガシーコードのリファクタリングや技術的負債の解消に主体的に取り組んでいます。また Claude Code をはじめとする生成AIを設計・実装・レビュー・テスト工程へ積極導入し、自分の効率化に留まらず、チーム共通のコマンド・プラグインを設計・展開することで、組織横断での開発生産性向上にも貢献しています。

今後はフロントエンドを主軸としつつ、バックエンド領域（Go等）にも実装経験を広げ、プロダクト全体に対して技術視点・事業視点の両面から価値提供できるエンジニアを目指しています。
